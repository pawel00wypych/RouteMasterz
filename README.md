# RouteMasterz

Logistic application to plan routes for drivers.
- Interactive map
- Logistic and Driver views

## How to run application?

- install node.js
- install docker
- clone repository
- run docker
- run "docker-compose build" in project folder
- run "docker-compose up" in project folder
